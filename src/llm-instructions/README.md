# Инструкции для LLM

Этот каталог содержит текстовые файлы с инструкциями для LLM, используемых в pipeline проекта bank-products-static.

## Доступные инструкции:

1. **verify-current.txt** - Проверка актуальности информации
   - Назначение: Проверка соответствия параметров продукта официальной информации на сайте банка
   - Вход: JSON продукта, дата обновления, источник
   - Выход: JSON с результатами проверки актуальности

2. **verify-credible.txt** - Проверка достоверности информации
   - Назначение: Проверка логической согласованности параметров и надежности источника
   - Вход: JSON продукта, источник информации
   - Выход: JSON с результатами проверки достоверности

3. **validate-format.txt** - Форматирование и валидация JSON
   - Назначение: Проверка структуры и формата JSON файла
   - Вход: JSON продукта
   - Выход: JSON с результатами проверки формата

4. **generate-descriptions.txt** - Генерация описаний и метаданных
   - Назначение: Создание описаний и SEO-метаданных для продукта
   - Вход: JSON продукта (без описаний)
   - Выход: JSON с сгенерированными описаниями и метаданными

5. **verify-type.txt** - Проверка соответствия типу продукта
   - Назначение: Проверка соответствия параметров продукта заявленному типу
   - Вход: JSON продукта, заявленный тип
   - Выход: JSON с результатами проверки типа

6. **check-duplicates.txt** - Проверка на дубликаты
   - Назначение: Определение, является ли продукт дубликатом существующего
   - Вход: JSON нового продукта, список существующих продуктов
   - Выход: JSON с результатами проверки на дубликаты
