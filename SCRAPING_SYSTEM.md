# Система автоматического сбора данных о банковских продуктах

## Обзор

Система автоматического сбора данных предназначена для регулярного сканирования официальных сайтов банков с целью получения актуальной информации о банковских продуктах (ипотека, кредиты, вклады).

## Структура

Система состоит из следующих компонентов:

- `bank-scraper.js` - основной скрипт для сканирования банковских сайтов
- `product-parser.js` - модуль для парсинга HTML-страниц и извлечения информации о продуктах
- `process-scraping-results.js` - скрипт для обработки результатов сканирования и сохранения в JSON-файлы

## Запуск скрапинга

Для запуска скрапинга выполните команду:

```bash
npm run scrape
```

## Конфигурация

Конфигурация скрапинга находится в файле `bank-scraper.js` в объекте `scrapingConfig`. Там можно настроить:

- Таймауты запросов
- HTTP заголовки
- Задержки между запросами
- Список банков и их веб-сайтов

## Обработка результатов

Результаты скрапинга сохраняются в JSON-файлы в соответствии с архитектурой проекта:

```
src/
└── data/
    └── pdfs/
        ├── mortgage/
        │   └── [bank-name]/
        ├── credits/
        │   └── [bank-name]/
        └── deposits/
            └── [bank-name]/
```

## Ограничения

- Некоторые банковские сайты могут блокировать автоматический доступ
- Структура веб-страниц может меняться, требуя обновления парсеров
- Для некоторых банков может потребоваться специальная аутентификация
